About Me

Perkenalkan nama saya Vildan Vinanda, lahir di Sumedang 30 April 2000. Kegiatan sehari-hari saya adalah nongkrong di depan laptop untuk belajar mengenai pemrograman android dan ketika saya memiliki waktu luang saya suka menyempatkan diri untuk membuat sebuah video yang nantinya dijadkan konten Youtube saya disamping itu saya juga suka berinteraksi dengan hewan peliharaan saya dan membantu pekerjaan ayah.

Saya merupakan anak ke 2 dari 2 bersaudara, ayah kami merupakan seorang pensiunan guru dan sekarang berprofesi sebagai peternak, sedangkan ibu kami berprofesi sebagai pedagang yang memiliki warung kecil kecilan yang menjajakan makanan anak. Hobi saya badminton dan basket, namun untuk hobi diluar yang berkaitan dengan fisik saya dari dulu suka nongkrong di depan komputer/laptop karena disana banyak sekali hal yang dapat saya lakukan dan dapat mencurahkan semua ide saya dan dari sana juga saya bisa dapat uang dengan menawarkan jasa desain grafis dll, Oleh sebab itu, saya tertarik untuk memilih kuliah dengan program studi informatika di Universitas Sebelas April (UNSAP).

Saya mulai terjun ke dunia pemrograman android pada tahun 2022 dimana pada saat itu saya mendapatkan tugas dari dosen untuk membuat sebuah aplikasi, dan saya memutuskan untuk membuat aplikasi bebrbasis android. Dari mulai saat itu saya tertarik untuk lebih jauh terjun untuk mempelajari pemrograman android, untuk menambah pengalaman saya juga pernah malakukan magang di Kilau Indonesia cabang Sumedang selama -/+ 2 bulam di bagian IT yang lebih tepatnya disana saya berperan sebagai front-end developer android mengunakan react native dan juga gitlab untuk sharing code bersama tim lainnnya. Setelah magang saya juga mulai menawarkan jasa pembuatan aplikasi android dengan bahasa java kepada beberapa teman yang membutuhkan, karena saya mendapatkan uang yang tidak sedikit dimasa itu saya jadi tambah tertarik lagi untuk terjun lebih dalam di dunia pengembangan android ini dan ingin menambah pengalaman dibidang ini.

Berbekal pendidikan dan pengalaman yang saya tempuh, saya berharap dapat mengembangkan kemampuan dan karier saya sebagai android developer di perusahaan yang Anda pimpin.
