App Suketdes

Aplikasi ini merupaan project magang di kilau indonesia, aplikasi ini berfungsi untuk monitoring anak asuh pada lembaga kemanusiaan Kilau Indonesia.

Deskripsi

Project aplikasi ini diracang oleh beberapa individu yang tergabung kedalam tim mobile yang memiliki tugas masing-masing. Pada pembagian tugas saya ditugaskan untuk membantu mengerjakan bagian font-end (membuat code dari UI yang telah di buat oleh bagian UI/UX) bersama denga teman saya. Pada pengembangan aplikasi ini kami menggunakan metode agile karena metode ini dinilai sangat fleksibel dengan perubahan yang terjadi. Adapaun framework yang digunakan adalah React Native.

Untuk pengumpulan code yang telah disusun kami menggunakan gitlab dan juga copy paste manual via flash disk. program magang ini dilakukan selama 3 bulan.

Spesifikasi Aplikasi

RAM : 2 GB

Memory : < 1 GB;

OS : Androd (5.0 Lolipop);
